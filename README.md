# React To-Do List

Aplicação de **lista de tarefas (To-Do List)** desenvolvida em **React** com o objetivo de praticar conceitos fundamentais do desenvolvimento **Front-End moderno**, como gerenciamento de estado, renderização dinâmica de componentes e persistência de dados no navegador.

---

## 🚀 Tecnologias Utilizadas

* **React**
* **JavaScript**
* **HTML5**
* **CSS3**
* **Vite**

---

## 📚 Conceitos Aplicados

Este projeto foi desenvolvido para praticar alguns conceitos essenciais do ecossistema React:

* **useState** para controle de estado da aplicação
* **Renderização dinâmica com `map()`** para exibir a lista de tarefas
* **Persistência de dados com `LocalStorage`**, permitindo que as tarefas permaneçam salvas mesmo após atualizar ou fechar o navegador

---

## ✨ Funcionalidades

* ➕ (Adicionar) novas tarefas
* 📋 Visualizar lista de tarefas
* ❌ Remover tarefas
* 💾 Armazenamento das tarefas no navegador utilizando **LocalStorage**

---

## 🖥️ Interface

Aplicação simples e intuitiva, desenvolvida com foco na prática de lógica de programação e nos conceitos fundamentais do **React**.
